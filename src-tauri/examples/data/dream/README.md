# `dream` - example dataset

This directory contains a 2D-QSAR dataset with 37 samples and 408 descriptor features.

- Structure (`X.csv`): a `;`-separated matrix of dimensions 37x408 (plus a header row and an index column).
- Activity (`y.csv`): the pIC50 values for each matching row of X (one value per row)

This dataset is called "dream" because it once enabled a $Q^2 = 1$ model.
